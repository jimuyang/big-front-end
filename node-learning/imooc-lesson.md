### require 规则
* / 表示绝对路径  ./ 表示相对路径
* 支持 js json node 扩展名 不写依次尝试
* 不写路径认为是build-in模块 或者各级 mode_modules

### require 特性
* module 被加载的时候执行 之后缓存不重复加载
* 循环加载时 只输出已执行的部分 还未执行的部分不会输出
